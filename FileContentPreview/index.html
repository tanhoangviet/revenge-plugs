What changes in chat.

The plugin turns raw text attachments into a readable, configurable preview flow.

Reader

Open code without leaving Discord.

Line numbers, word wrap, monospace mode, chunk loading, and a status bar keep long files readable.

Highlight

Syntax colors and file metrics.

Keywords, strings, comments, imports, and blocks are surfaced directly in the preview.

Explain

Local code explanation.

The info panel summarizes what the file appears to do without sending code to a server.

Configure

Theme, glass, motion, and reader defaults.

Choose VS Code-style themes, transparent preview panels, glass zoom timing, chunk size, and default reader behavior.
