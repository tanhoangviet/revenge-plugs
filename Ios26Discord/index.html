Dynamic Island
iOS 26 Discord Liquid glass preview surface

What changes in chat.

The plugin adds a liquid glass visual layer across Discord while turning Dynamic Island into a compact gesture surface.

Island

Smaller island, no duplicate overlays.

The island now mounts through one runtime host and sits lower near the top bar with a slimmer collapsed state.

Gestures

Pull and swipe search actions.

Pull down for message search, swipe right for user search, and swipe left for server search with live arrow hints.

Commands

Fast slash command panel.

Tap the island to open an input, search buttons, and quick slash command chips.

Runtime

More reliable overlay patching.

The island now patches normal React and JSX runtime paths, plus diagnostics for Evaluate JavaScript.
